HTML5的优势：
	1.跨平台
		所谓跨平台指的是做出来的东西能够适应各种浏览器，各种设备，各种系统（android，ios，swift），还能适应不同的屏幕大小。
	2.快速迭代
		HTML开发应用的效率远远快于原生应用
	3.降低成本
		目前，只要不是过于庞大的App，HTML能做出与原声几乎没差别的效果，成本会比原生要少很多。
	4.导流入口多
		HTML5的到导流入口非常多，比如超级APP（微信朋友圈），
		搜索引擎，应用市场，浏览器等都可以作为HTM5的流量入口。

	HTML5会成为未来网站发展的新标准
	现在的HTML5技术，不仅仅可以开发网站页面，还能开发wed app,oa自动办公系统等，微信开发等其他项目
	不仅如此，前端现在逐渐开始向后台侵蚀，其js的特性node.js。可以实现后台开发。
